One-shot learning

Q. What is a typical application of one-shot learning? A. Facial recognition

Q. Which problem does one-shot learning solve? A. Binary distinguishment of unique classes (eg. are these two pictures of the same person?)

Q. In which form must data for one-shot learning be presented? A. Triplets (Anchor, Positive, Negative), where positive means “same class as anchor”
