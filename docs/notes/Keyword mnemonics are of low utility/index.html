Keyword mnemonics are of low utility

Q. What is the procedure for using keyword mnemonics? A. Find a proxy that sounds like the word, then make an action with that proxy that explains the word.

Q. What is a paradoxical effect of using keyword mnemonics? A. They can accelerate encoding, but also forgetting.

Q. When are keyword mnemonics useful? A. If you need to retain the information for hours to a day and can't write it down (ie. rarely).
