# Python
Q. How do classes relate to objects?
A. Classes are templates for objects

Q. How do you define a class in Python?
A. 
``` py
# Dog class
class Dog:
  # Method of the class
  def bark(self):
    print("Ham-Ham")
```

Q. How do you create an instance of Dog?

```py
# Dog class
class Dog:
  # Method of the class
  def bark(self):
    print("Ham-Ham")
```
A. 
```py
# Create a new instance
charlie = Dog()
```
